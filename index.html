MouseDancer

Keeps your mouse dancing so your screen never sleeps.

✨ What it does

Automatically moves your mouse cursor at regular intervals — no more idle screen, no more away status.

⚡ Requirements (Python version)

Python 3 + pyautogui

pip install pyautogui

🚀 How to run (Python version)

python move_mouse.py
