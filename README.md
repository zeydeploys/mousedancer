# 🐭 MouseDancer

Keeps your mouse dancing so your screen never sleeps.

## What it does
Automatically moves your mouse cursor at regular intervals to prevent your screen from going idle or your status from showing "away".

## How to run
```bash
python move_mouse.py
```

## Requirements
- Python 3
- `pyautogui` library

## Install dependencies
```bash
pip install pyautogui
```
